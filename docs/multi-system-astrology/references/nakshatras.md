# Nakshatra Reference Guide

Complete reference for all 27 Nakshatras (Lunar Mansions) used in Vedic Astrology.

## Overview

Each Nakshatra spans 13°20' (13.333...°) of the zodiac. The Moon traverses all 27 Nakshatras in approximately 27.3 days.

### Calculation Formula

```
Nakshatra Index = floor(Sidereal Moon Longitude / 13.3333)
Pada = floor((Moon Longitude mod 13.3333) / 3.3333) + 1
```

## Nakshatra Compatibility (Nadi)

For marriage compatibility, Nakshatras are grouped into three Nadis:
- **Adi (Beginning):** Ashwini, Ardra, Punarvasu, Uttara Phalguni, Hasta, Jyeshtha, Mula, Shatabhisha, Purva Bhadrapada
- **Madhya (Middle):** Bharani, Mrigashira, Pushya, Purva Phalguni, Chitra, Anuradha, Purva Ashadha, Dhanishtha, Uttara Bhadrapada
- **Antya (End):** Krittika, Rohini, Ashlesha, Magha, Swati, Vishakha, Uttara Ashadha, Shravana, Revati

Same Nadi = 0 points (Nadi Dosha)
Different Nadi = 8 points (Best compatibility)

## Vimshottari Dasha Sequence

Nakshatra rulers determine the starting Mahadasha:
1. Ketu (7 years) - Ashwini, Magha, Mula
2. Venus (20 years) - Bharani, Purva Phalguni, Purva Ashadha
3. Sun (6 years) - Krittika, Uttara Phalguni, Uttara Ashadha
4. Moon (10 years) - Rohini, Hasta, Shravana
5. Mars (7 years) - Mrigashira, Chitra, Dhanishtha
6. Rahu (18 years) - Ardra, Swati, Shatabhisha
7. Jupiter (16 years) - Punarvasu, Vishakha, Purva Bhadrapada
8. Saturn (19 years) - Pushya, Anuradha, Uttara Bhadrapada
9. Mercury (17 years) - Ashlesha, Jyeshtha, Revati

Total cycle: 120 years
